# Course project — Spectre / speculative-execution side channels (handouts)

## Problem

The course project required selecting a top-venue security paper (CCS / S&P /
USENIX / NDSS, post-2013, with significant citations), recording a presentation,
and writing a short (~500-word) proposal covering the area, problem, solution,
evaluation, and key takeaway.

My chosen topic was **Spectre** and micro-architectural side-channel attacks —
speculative execution that leaks data across security boundaries through
micro-architectural state (e.g. the cache).
